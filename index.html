Sobre Mí

Soy un desarrollador apasionado por crear soluciones innovadoras y funcionales. Mi experiencia incluye el desarrollo de proyectos como:

  • Kanthus Project: Sitio web para una hamburguesería, desarrollado con Vanilla JavaScript y Node.js.
  • Clon de Airbnb: Plataforma web creada utilizando Flask para la gestión de alojamientos.
